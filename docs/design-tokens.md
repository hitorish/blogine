# 디자인 토큰

common.css의 `:root`에 정의된 CSS 변수와 디자인 규칙입니다.

---

## 브랜드 컬러 (교보생명 기준)

```css
:root {
  --kyobo-green:       #059669;   /* 메인 브랜드 그린 */
  --kyobo-green-light: #34d399;   /* 밝은 그린 (강조, 프로그레스) */
  --kyobo-green-mid:   #10b981;   /* 중간 그린 (그라데이션) */
  --kyobo-green-dark:  #047857;   /* 어두운 그린 (배경) */
  --kyobo-green-bg:    #f0fdf4;   /* 연한 그린 배경 */
  --kyobo-green-bg2:   #ecfdf5;   /* 연한 그린 배경 2 */
  --kyobo-green-bg3:   #d1fae5;   /* 연한 그린 배경 3 */
  --kyobo-gold:        #d4a853;   /* 골드 (미사용, 예비) */
  --text-dark:         #1f2937;   /* 본문 텍스트 */
  --text-sub:          #4b5563;   /* 서브 텍스트 */
  --text-light:        #9ca3af;   /* 각주, 출처 */
}
```

### 다른 브랜드 적용 시

`:root` 변수만 교체하면 전체 색상이 변경됩니다. 추가로 수정이 필요한 곳:

| 위치 | 하드코딩된 색상 | 용도 |
|------|----------------|------|
| `scene-1.css` | 배경 그라데이션 | 인트로 분위기 |
| `scene-9.css` | 배경 그라데이션 (`#047857 → #10b981`) | 브랜드 씬 |
| `common.css` | `#34d399` (프로그레스, 파티클) | UI 요소 |
| inline `style` | `color:#059669`, `color:#dc2626` | 텍스트 강조 |

## 타이포그래피

### 폰트

```css
font-family: 'Noto Sans KR', sans-serif;
```

Google Fonts에서 로드합니다. 사용 weight: 400, 500, 700, 900.

### 텍스트 크기 체계

모든 크기는 `clamp(min, preferred, max)` 반응형입니다.

| 용도 | 크기 | weight |
|------|------|--------|
| 인트로 타이틀 | `clamp(62px, 14.4vw, 98px)` | 900 |
| 씬 헤딩 | `clamp(55px, 13.2vw, 86px)` | 900 |
| 본문 텍스트 | `clamp(38px, 8.4vw, 55px)` | 500 |
| 하이라이트 박스 | `clamp(34px, 7.2vw, 48px)` | 700 |
| 서브 카피 | `clamp(41px, 9.6vw, 65px)` | 700 |
| 각주/출처 | `clamp(26px, 5.4vw, 36px)` | 400 |
| 자막 | `clamp(20px, 4.2vw, 26px)` | 500 |

### 행간 (line-height)

| 용도 | 값 |
|------|-----|
| 타이틀/헤딩 | 1.3 |
| 본문 | 1.7 |
| 자막 | 1.8 |

## 레이아웃

### 영상 비율

- **9:16** (세로형, 인스타 릴스/유튜브 숏츠)
- 캡처 해상도: **720 x 1280**

### 씬 패딩

| 클래스 | padding-top | padding-bottom | 용도 |
|--------|------------|---------------|------|
| `.scene` | 5% | 30% | 기본 (컨트롤 영역 확보) |
| `.scene.extended` | 3% | 18% | 콘텐츠 많은 씬 |

### 컨텐츠 영역

| 클래스 | 너비 | max-width |
|--------|------|-----------|
| `.content-area` | 60% | 432px |
| `.content-area-wide` | 70% | 504px |

## 씬 배경 패턴

| 씬 유형 | 배경 | 예시 |
|---------|------|------|
| 인트로 | 어두운 그라데이션 + 사진 | `linear-gradient(160deg, #1a0a0a, #3d1111, #0d1a0d)` |
| 설명 씬 | 밝은 단색 | `#F1F1F1` |
| 강조 씬 | 브랜드 그라데이션 | `linear-gradient(160deg, #047857, #059669, #10b981)` |
| 아웃트로 | 어두운 배경 + 오버레이 | 커스텀 |
| 브랜드 | 브랜드 그린 그라데이션 | `linear-gradient(160deg, #047857, #10b981)` |
